## 🏗️ 素材库怎么用

### 第 1 步:创建素材库目录

默认在 `./assets_library/`,可以在 AI 面板第四行改。

### 第 2 步:放参考图,文件名就是关键字

```
assets_library/
├── 林默.png          ← 关键字: "林默"
├── 苏郁.jpg          ← 关键字: "苏郁"
├── 蔷薇宫.png        ← 关键字: "蔷薇宫"
├── 雨夜巷子.png      ← 关键字: "雨夜巷子"
└── aliases.json     ← 可选,别名配置
```

### 第 3 步(可选):配别名 `aliases.json`

```json
{
  "林默": ["林渊", "主角", "渊哥"],
  "苏郁": ["苏老板", "苏女神"]
}
```
这样说「渊哥去找苏老板」也能匹配到两张图。

### 第 4 步:点「🔄 扫描」

工具顶部 AI 面板里,点一下就索引好了。

### 第 5 步:发 GPT 时自动用

1. 打开 GPT 助手面板
2. 输入框写:`苏郁走进蔷薇宫,向林默问好,生成国漫条漫风格的画面`
3. 下方会提示:`✅ 3 张:苏郁, 蔷薇宫, 林默`
4. 点 **「🚀 一键 上传+发送」** — 三张图会一起上传,然后文本一起发出去

## 📚 提示词库(手动导入/编辑)

### GPT 提示词库(`gpt_prompts.json`)
首次启动自动生成,在当前目录。默认几条:
- 让 GPT 生成分镜首帧
- 让 GPT 生成分镜尾帧
- 人物设定图(三视图)
- 场景概念图

**使用:** GPT 面板的「📚 常用提示词」区域,**双击任一条**:
- 「是」= 替换当前输入框
- 「否」= 追加到末尾
- 「取消」= 不动

**管理:**
- ➕ 新增
- ✏️ 编辑
- ❌ 删除
- 📂 打开 JSON 文件手动编辑

### Wan 2.2 提示词库(`wan22_prompts.json`)
任务列表上方新增「**📖 Wan 提示词库**」按钮。默认提示词:
- 镜头推近
- 动作爆发
- 情感慢镜
- 时空过渡
- 惊醒
- 决心

**使用:** 点「📖 Wan 提示词库」→ 选中任务列表的场景 → 回到对话框 → 双击一条提示词 → 自动应用到选中的所有场景。

## 🔧 工作原理

### 素材匹配(贪心算法)
1. 读目录里的所有图片,文件名(去扩展名)当关键字
2. 读 `aliases.json`,建立别名映射
3. 匹配时**按关键字长度降序**扫描(避免「苏郁」被「苏」截胡)
4. 匹到一个就在输入文本里挖空,防重复

### 图片上传(Selenium send_keys)
三级兜底:
1. **方案 A**:找现成的 `<input type="file">` — 直接塞路径
2. **方案 B**:点 `+` 按钮(`[data-testid="composer-plus-btn"]`)触发显示 input
3. **方案 C**:JS 新建一个 `<input type="file">` 挂到 body — 再塞路径

好处:**绕过系统文件对话框**,无缝连贯,支持多图。

## ❓ 常见问题

**Q:匹配到图但没上传成功?**
A:先手动在 GPT 网页上点一下 + 按钮,确认上传功能正常。再回来自动上传。某些镜像站的上传 DOM 可能不同,需要等几秒让 input 出现。

**Q:AI 生成的场景名不是我素材库的关键字?**
A:两种办法:① 把素材库文件名改成 AI 常用的那种;② 配 `aliases.json` 把常见词映射到素材名。

**Q:提示词库想备份?**
A:把当前目录的 `gpt_prompts.json` 和 `wan22_prompts.json` 拷走就行。

**Q:Wan 提示词应用到场景后可以改吗?**
A:可以。双击场景行 → 编辑 → 改 prompt 字段 → 保存。

## 🚀 推荐工作流

```
┌─ 素材库准备(一次性)
│  1. 在 assets_library/ 放角色图,文件名=关键字
│  2. 编辑 aliases.json 配常用别名
│  3. 点「🔄 扫描」
│
├─ 生成阶段
│  1. 「📚 小说一键改分镜」→ 填入场景 + AI 提示词
│  2. 「🌐 打开 GPT」→ 输入「生成XXX场景的首帧图」
│  3. GPT 面板自动匹配素材 → 🚀 一键上传+发送
│  4. GPT 回复后 → 🖼 下载图片 → 作为场景首帧
│  5. 重复生成尾帧
│
└─ 批量视频
   1. 每个场景都有首尾帧图 + 动作指令
   2. 「🚀 开始批量生成」→ 等 Wan 2.2 跑完
   3. 双击场景行预览视频
```
